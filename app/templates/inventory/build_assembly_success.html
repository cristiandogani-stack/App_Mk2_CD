{# Success page for completing an assembly build inside a production box modal. When an assembly is built from within a modal (embedded flag true), the backend returns this minimal page instead of redirecting back to the production box. The page contains a small JavaScript snippet that instructs the parent window to close the build modal and reload the list view so that the box status is refreshed immediately. Outside of embedded mode this template should not be used; the ``build_assembly`` view continues to redirect to the appropriate back_url when the build happens outside of a production box. #} Assemblaggio completato
